# Library Management Demo (Frontend)

A minimal JavaScript/HTML demo simulating a Library Management System.

## Features
- Add books and members
- Borrow and return books
- Simple interactive UI

## Deploy on Vercel
1. Push this repo to GitHub.
2. Go to [vercel.com](https://vercel.com) and import the repo.
3. Deploy → you’ll get a live URL like `https://library-demo.vercel.app`.

## Showcase
This demo simulates the logic of the original Java console project in a browser-friendly way.
